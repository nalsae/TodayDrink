# 🍹당신을 위한 칵테일 아카이브, 오늘 한 잔🍹

## 🔸 프로젝트 설명
다양하고 복잡한 칵테일에 대한 정보와 사용자 맞춤형 칵테일의 추천을 제공 받을 수 있는 사이트를 기획 및 제작하였습니다.

## 🔸 개발 단계
화면 설계 > DB 구축 > 디자인 > 페이지 제작 > DB 연동 > 배포 및 테스트

## 🔸 구현 기능
1. 칵테일 이름 검색(한글/영문)
2. 칵테일 테스트 및 추천
3. 칵테일 상세 정보 제공
4. 카테고리별 칵테일 분류
5. 상황별 칵테일 추천
6. 주변 칵테일 바 검색

## 🔸 기술 스펙
* Front-End <br>
: HTML, CSS, JS, jQuery, Jinja, Bootstrap <br>
* Back-End <br> 
: Python, mongoDB <br>
* Distribution <br>
: AWS
